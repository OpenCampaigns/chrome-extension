# @opencampaigns/extension

The native User Client for routing decentralized ads into the browser effortlessly.

Replace algorithmic tracking ads with intent-driven, fully autonomous, completely open-source commercial content directly into your browsing experience. 

## Structure
Built on **Chrome Manifest V3** and Lit Web Components, the extension utilizes a background Service Worker to continuously poll Nostr relay servers, filtering active marketing campaigns strictly using the locally-stored Intents defined dynamically by the user!

It injects an embedded DOM proxy logic detecting `.opencampaigns-slot` objects, executing entirely serverlessly!
